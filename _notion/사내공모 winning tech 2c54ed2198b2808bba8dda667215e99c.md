# 사내공모 winning tech

지원사유:

저는 multimodal, vision 모델 개발을 통해 실제 환경에서 작동하는 인지 AI를 연구해온 경험을 바탕으로, 로봇이 오픈월드에서 스스로 주변을 이해하고 과업을 제안하는 로봇 인지 기술 개발에 기여하고자 지원했습니다. 연구 초기에는 Human Pose Estimation, Action Localization 등 시각 기반 이해 기술을 다뤘고, 석사 과정과 실무에서는 Transformer 기반 모델을 개발하며 복잡한 장면 이해 및 로버스트한 시각 표현 학습 경험을 쌓았습니다. 이후 Multimodal task에서 VLM/MLM 기반 Examodal 프로젝트에서 온디바이스에 적용가능한 크기의 vision-language reasoning, 영상, 오디오, 로그 기반 이벤트 감지 모델을 직접 설계, 구현했습니다. 모델 설계뿐 아니라 실제 가전 가전 로그, 영상 데이터 기반 PoC, 모델 최적화, on-device 경량화, API 설계까지 end-to-end로 주도한 경험을 통해, 로봇 환경에서의 실사용 가능하고, 실내 공간을 이해하며 사용자의 요구를 먼저 읽어내는 인지 모델을 만드는 일에 제 역량을 온전히 사용하고 싶어 지원하게 되었습니다.

기여 역량:

제가 가장 크게 기여할 수 있는 부분은 Language-aligned Vision Model을 로봇 환경에 실제로 적용 가능한 형태로 설계하고 최적화할 수 있는 경험 기반 역량입니다. VLM, Audio Tower 확장, Cross-modal Alignment를 연구하고, 내부 데이터 기반 instruction tuning, multimodal prompting 환경을 구축해왔습니다. 또한 Pose/Action 분야에서 여러 국제 학회(WACV, PR, NN 등) 발표 및 SOTA 수준 성과를 달성하며 시각 표현 학습에 대한 깊이 있는 이해를 확보했습니다. 업무에서sms on-device, edge 제약 속에서 안정적으로 작동하는 모델 경량화와 성능 튜닝을 수행해 실제 제품 환경을 고려한 개발 능력을 검증받았습니다. 이를 바탕으로 로봇이 실공간에서 마주하는 open-set 상황에 강한 vision-language 인지 모델 설계가 가능합니다. 또한 웹 개발 및 서비스 개발 경험은 로봇 인터랙션 UX를 설계할 때 사용자 관점을 포함한 판단을 돕습니다. 연구–개발–적용을 모두 경험한 실전형 엔지니어로서 로봇 비전 지능 개발에 즉시 기여하겠습니다.

# 상황인지

지원사유:

저는 연구와 실무를 모두 경험하며 실제 환경을 이해하고 작동하는 AI를 만들고자 지속적으로 도전해 왔습니다. 업무 중 Multimodal, Vision 기반 인지 모델을 개발하며 영상, 오디오, 로그를 결합한 이벤트 감지, 상황 인지 모델링, Edge 환경 모델 최적화 등을 수행했고, 이러한 경험은 로봇이 실내 환경을 인지하고 변화량을 해석하는 이번 직무와 높은 연관성이 있다고 판단했습니다. 대학원 연구에서는 Pose Estimation, Action Localization, Occlusion-aware Transformer 등 시계열 기반 인지 모델을 다수 개발했고 WACV, NN, PR 등 국제 학회에 논문을 발표하며 모델링 능력을 검증받았습니다. 이렇게 연구, 현장 모두에서 이러한 역량을 직접 구축해온 경험을 살려, 로봇선행연구소의 홈로봇 멀티모달 상황인지 기술 고도화에 기여하고 싶어 지원했습니다.

기여역량:

제가 팀에 기여할 수 있는 가장 큰 역량은 시각, 언어, 오디오 데이터를 통합해 상황을 해석하는 모델을 실제 환경에 구현한 경험입니다. 이벤트 감지 모델을 개발하며 데이터 파이프라인 설계부터 모델 tuning 및 배포까지 End-to-End로 수행했고, Examodal 프로젝트에서 VLM, LLM 기반 오디오 모달리티 확장을 수행했습니다. Vision 분야에서는 포즈, 액션, 3D 구조 복원 등 시계열 특징을 다루는 Transformer 기반 모델 연구를 수행했고, Occlusion, Scene ambiguity, Sparse label 문제를 해결하는 방법론을 직접 설계했습니다. 이는 로봇 환경에서 발생하는 비정형 데이터, 가림, 시점 변화 등 다양한 문제를 해결하는 데 바로 적용 가능합니다. 또한 Edge, 가전 플랫폼에서 모델 최적화 및 경량화를 경험해 로봇 HW 제약 내에서 동작 가능한 모델 개발에도 강점이 있습니다. 문제를 구조화하는 능력, 데이터 수집/정제 능력, 빠른 실험 반복 속도, 그리고 사용자, 현장 관점을 고려한 실용적 설계 경험을 바탕으로 홈로봇의 상황인지, 추론 모델 고도화에 즉시 기여하겠습니다.
